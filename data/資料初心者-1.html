Chapter 2 資料初心者

2.1 安装 R 和 RStudio

要使用 R Markdown，首先需要安装 R 和 RStudio。 R語言是一種自由軟體程式語言，主要用於資料分析與統計運算，而RStudio是R語言的IDE，提供一般桌面板與伺服器版 能夠在官方網站進行下載

https://www.r-project.org/

https://www.rstudio.com/

2.2 建立新專案

打開 RStudio，然後在控制台（Console）中輸入命令來安裝 bookdown

install.packages("bookdown")

創建一個新的 bookdown 專案 在 RStudio 中，選擇 File > New Project。 選擇 New Directory，然後選擇 Book Project using bookdown。 輸入專案的名稱和存儲位置。 點擊 Create Project。

2.3 R markdown

  1. 在{r} code區段時，起初常常無法編織成功，需要注意代碼塊echo、message、warning 等是否正確
